What is an Agent Permit Card?

An Agent Permit Card is a simple permission profile for an AI agent.

It shows:

  • What the agent can do
  • What requires human approval
  • What the agent must never do
  • What data the agent can access
  • What actions must be logged

Before an AI agent acts, it should carry a permit card.

Example: Email Assistant

A simple permission card for an AI agent that helps with incoming emails.

Agent Permit Card
Email Assistant

🟢 Can do

  • Read incoming emails
  • Draft replies
  • Summarize attachments

🟡 Needs approval

  • Send an email
  • Forward a document
  • Update customer records

🔴 Cannot do

  • Delete emails
  • Access payment data
  • Send legal or financial statements

🔵 Data access

  • Email content
  • Sender name and address
  • Attachments marked as safe

⚫ Logs

  • Every draft is saved
  • Every approved action is recorded
  • Every rejected action is tracked

Why it matters

AI agents are starting to access real tools: email, files, calendars, CRMs, databases, and internal workflows.

Without clear permissions, teams cannot easily answer:

  • What did this agent do?
  • Was it allowed?
  • Who approved it?
  • Can we audit it later?

Agent Permit Card makes AI agent permissions easy to understand before anything goes wrong.
